Inicios

En 2022 comencé oficialmente mi camino en el mundo de la programación y desarrollo IT. Desde entonces, estoy constantemente realizado bootcamps y estudios autodidactas complementarios para poder incursionar profesionalmente en esta área. Realizo proyectos desarrollando aplicaciones web y Apis buscando soluciones para usuarios de distintos rubros y usos "típicos", con tecnologías como Node.js, React.js, Tailwindcss, Javascript para la interfaz del usuario, y del lado del Backend, Node.js, Express, Javasctipt, Vite como empaquetador y Mysql como gestor de base de datos. Para el despliegue suelo utilizar Netlify para el Frontend, y Railway para el Backend para pruebas o desarrollo. En otros entornos mas estructurados utilizo AWS para alojar toda la arquitectura de las aplicaciones, en servicios como S3, EC2, bases de datos RDS, o en ocasiones Elastic Beanstalk. Me considero una persona muy capaz, responsable, y con habilidades sólidas adquiridas en mis experiencias de trabajos las cuales fueron todas en equipo. Adaptabilidad, Compañerismo, Soluciones en equipo, anticiparse a posibles fallas, etc.

Mis Skills: ReactJS, NodeJS, JavaScript, Tailwindcss, Git & GitHub, MySQL.
